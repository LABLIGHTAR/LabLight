TITLE: Adding UI Toolkit Assembly References in .asmdef (JSON)
DESCRIPTION: This JSON snippet demonstrates how to configure a Unity assembly definition file (.asmdef) to explicitly reference the UnityEngine.UIElementsModule and UnityEditor.UIElementsModule assemblies. This is required when using features exclusive to the UI Toolkit package to ensure correct compilation order and avoid missing API errors.
SOURCE: https://github.com/needle-mirror/com.unity.ui/blob/master/README.md#_snippet_0

LANGUAGE: JSON
CODE:
```
{
  "name": "AssemblyConsumingNewAPI",
  "references": [
     "UnityEngine.UIElementsModule",
     "UnityEditor.UIElementsModule"
     ],
  "includePlatforms": [],
  "excludePlatforms": [],
  "allowUnsafeCode": true,
  "overrideReferences": false,
  "precompiledReferences": [],
  "autoReferenced": false,
  "defineConstraints": [],
  "versionDefines": [],
  "noEngineReferences": false
}
```
